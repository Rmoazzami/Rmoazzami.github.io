The Story Behind Some of My Favorite Projects

Atlanta Laughs

Recently, I've been creating a website called Atlanta Laughs. Since I've been doing comedy in Atlanta, I realized there is a big disparity in which the talent of the local comics greatly exceeds the communities awareness of it. After throwing my first show in early 2019, I noticed that audience members went up to comics and asked them when their next upcoming shows were, with comics shrugging, not knowing themselves. This is when the idea for Atlanta Laughs was born.

Currently, the project consists of two parts. The first is a texting service engineered in Twilio, AWS Lambdas, AWS DynamoDB, and AWS API Gateway that allows anyone subscribed to recieve a text message every Sunday 6PM of the best shows coming up that week (Text Subscribe to (219)-244-4231 if you want to be apart of the action). Responding with the name of the show you want to see triggers another text in which you'll recieve more information such as the location of the show, time, price, lineup, and parking.

The second aspect of this project consists of a website where Atlanta Laughs, local comics, and shows are promoted. This website is currently being developed in ReactJs, Flask, Express.JS, and Node.JS. I'm still in the process of acquiring AtlantaLaughs.com but that's where you'll be able to find it when it's up and running! For now if you want to keep up you can follow Atlanta_Laughs on instagram!

AlgoTrading

Two summers ago I became really interested in the stock market and the general idea of investing. I delved into some popular investing books to see what I could learn for myself. I ended up reading famous books such as Benjamin Graham's 'The Intelligent Investor', Peter Lynch's 'One Up on Wall Street', and Lawrence McMillan's 'Options as a Strategic Investment'.

I realized most people's strategies often involved trading when a set of parameters are met, for example if the price of the stock hits the upper bound of a Bollinger Band you sell or if it hits the lower bound you sell. Since these parameters could be easily identified algorithmically, it was a perfect time to put my computer science skills to the test

My first rendition of my algorithm trader was created at Georgia Tech's 2018 Hackathon. I used Python Numpy Arrays and Pandas to pull in stock prices from IEX Finance and indicators from Goldman Sachs' Marquee API to create a graphical representation of Bollinger Bands coupled with the Goldman Sachs' indicators given the stock ticker a user put in. The Bollinger Bands allowed the user to see if the stock price was currently being under or over valued on the market place while the Goldman Sachs indicators allowed the user to see the overall health of the company, thus allowing users to make easy and informed decisions.

In the next rendition of my algorithm trader I wanted to incorporate news data an important indicator. I assembled a team of three for the 2018 Vanderbilt Hackathon and during the course of this period we set up a neural network that was fed in sentiment scores derived from the Natural Language Toolkit used on news articles on the top 500 companies from the Reuters API and technicial indicators such as RSI, Stoch, and Aroon from the Alpha Vantage API.

The Cyber Savant project was an artificial intelligence aimed to assign real-time suspicion levels to network traffic. Previously the AI only had static batch analysis with users having to manually put in a snapshot of a given time period of network traffic as a CSV file. I engineered a Java pipeline that would streamline and parse live network data from Kafka and Redis instances and be filtered by a YAML Filter function.

An experiment was held to test this concept. Participants were asked to pick pictures of Bees and Not Bees to help retrain a model. Asserting that if participants were able to pick pictures that helped retrain a model above its previous accuracy scores they understood at a top level how an AI operated and their mental model was sound. My role on this project was helping develop the experiment.
